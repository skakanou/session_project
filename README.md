
# 📌 Event Booking App

## 📖 Description

Cette application est un système de réservation d’événements développé avec Flask. Elle permet aux utilisateurs de consulter une liste d’événements disponibles, de rechercher un événement, puis de réserver un nombre de places. Chaque utilisateur peut également consulter et gérer ses réservations depuis un tableau de bord personnel.

Une interface administrateur permet de créer, modifier et supprimer des événements.

---

## 🎯 Fonctionnalités

### 👤 Utilisateur

* Consulter la liste des événements
* Rechercher un événement par nom ou lieu
* Réserver un événement avec un nombre de places choisi
* Consulter ses réservations
* Annuler une réservation

### 🛠️ Administrateur

* Ajouter de nouveaux événements
* Modifier les événements existants
* Supprimer des événements
* Gérer les places disponibles

---

## ⚙️ Fonctionnement

* Lorsqu’un utilisateur réserve un événement, le nombre de places disponibles est automatiquement mis à jour.
* Chaque réservation est liée à un utilisateur et à un événement.
* Les utilisateurs peuvent réserver plusieurs places en une seule action.
* Les réservations peuvent être annulées depuis le tableau de bord.

---

## 🚀 Objectif

Ce projet simule une plateforme de réservation d’événements simple et fonctionnelle, inspirée de systèmes comme Eventbrite, en mettant en pratique les concepts de développement web avec Flask et une base de données relationnelle.
